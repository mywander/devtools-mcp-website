# DevFest Workshop — Chrome DevTools MCP Server (示例站点)

本项目是一个静态演示页面，用于记录李良普参加的 Google DevFest Workshop（主题：Chrome DevTools MCP Server）。风格为像素艺术，使用 `Press Start 2P` 字体作为像素字形。

快速预览：

在项目目录下运行一个静态服务器，例如使用 Python（macOS 自带）：

```bash
cd /Users/liliangpu/Downloads/javascript/mcp/devtools-mcp-website
python3 -m http.server 8000
# 然后在浏览器打开 http://localhost:8000
```
